# OrangeHRM Automation Framework

This project is a **Selenium Automation Framework** built for the OrangeHRM application using **Java, Selenium WebDriver, Cucumber BDD, JUnit, and Page Object Model (POM)**.

## Tech Stack
- Java
- Selenium WebDriver
- Cucumber BDD
- JUnit
- Maven
- Page Object Model (POM)
- Apache POI (Excel data handling)

## Framework Features
- Page Object Model design pattern
- BDD test scenarios using Cucumber
- Data-driven testing using Excel (Apache POI)
- Reusable utility classes
- Explicit waits for stable test execution
- Generic methods for extracting UI table data

## Test Scenarios Covered
- Login functionality
- Admin user creation
- User list validation
- Export user list data into Excel
- Toast message validation

## Project Structure
src/test/java
│
├── pages
├── stepdefinitions
├── utilities
├── runners


## How to Run Tests

1. Clone the repository
2. Open project in IDE (IntelliJ / Eclipse)
3. Run the Cucumber Runner class
